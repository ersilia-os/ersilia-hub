## Dependencies ##

- nodejs v22+
- npm 10.9+
- [Angular CLI v17](https://github.com/angular/angular-cli)

## Execution ##

There is no custom requirements for local development, we simply run `ng serve`.\
To keep it future-proof, we created a wrapper script [./operations/serve-pwa.sh](../operations/serve-pwa.sh).

```
./operations/serve-pwa.sh
```
